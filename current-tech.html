Current AI Technologies in STEM Education

An overview of existing AI tools and platforms that are currently being used in STEM education.

Current AI Tools in Education

The integration of AI in STEM education is rapidly evolving, with new tools and systems releasing constantly. These technologies range from general large language models that have a large variety of knowledge to specialized educational systems designed specifically for learning. While some schools embrace these tools and teach students how to apropriately use them, others are being used by students behind the scenes. Below is a list of a some of the most common avaible resources and their key features that attract students. As new technology continues to come out, it will remain a constant battle between students and schools for the intgration of AI within the education system.

ChatGPT

OpenAI's conversational AI that students and educators use for explanations, problem-solving assistance, and content generation. It can help with homework, explain complex concepts, and generate practice problems.

Key Features:

  • Natural language interaction
  • Multi-domain knowledge
  • Code generation
  • Explanation of concepts

Khanmigo

Khan Academy's AI-powered tutor that provides personalized learning experiences. It guides students through problems with Socratic questioning rather than giving direct answers, promoting deeper understanding.

Key Features:

  • Socratic method tutoring
  • Personalized learning paths
  • Real-time feedback
  • Progress tracking

GitHub Copilot

An AI pair programmer that suggests code completions and entire functions as developers type. Widely used in computer science education for helping students learn programming patterns and best practices.

Key Features:

  • Code autocompletion
  • Function generation
  • Multi-language support
  • Learning from context

AI Tutoring Systems

Various platforms like Carnegie Learning, DreamBox, and others that use AI to adapt instruction to individual student needs, providing customized problem sets and explanations based on student performance.

Key Features:

  • Adaptive difficulty
  • Individualized pacing
  • Misconception detection
  • Performance analytics
